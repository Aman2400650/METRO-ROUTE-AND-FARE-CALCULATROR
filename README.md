# 🚇 Metro Route & Fare Calculator

A C++ based console application that simulates the Delhi Metro system and calculates the shortest route, travel time, and fare between stations using graph algorithms.

---

## ✨ Features
- 📍 Find shortest path between two metro stations  
- ⏱️ Calculates minimum travel time  
- 💰 Computes total fare  
- 🧠 Uses Dijkstra’s Algorithm (Graph-based approach)  
- 🖥️ Simple CLI (Command Line Interface)

---

## 🛠️ Tech Stack
- Language: C++  
- Concepts: Graphs, Dijkstra Algorithm, OOP  

---

## 📂 Project Structure
- `main.cpp` → User interaction & execution  
- `metro.h` → Class declarations  
- `metro.cpp` → Logic & algorithm implementation  

---

g++ main.cpp metro.cpp -o main
